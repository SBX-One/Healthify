# Healthify

Healthify is a web platform that provides trusted medical information, including health blogs, nearby pharmacies and hospitals, as well as details about diseases and medications. It aims to make healthcare knowledge easily accessible and help users make informed health decisions.

## Demo / Live Preview

You can access the live version of this project here:  
https://healthify-cyan.vercel.app/


## Technologies Used

Here are the main technologies used in this project:

- React JS (v 18.3.1) - A popular JavaScript library for building dynamic and interactive user interfaces using component-based architecture.

- Tailwind CSS (v 3.4.17) - Tailwind CSS (v3.4.17) – A utility-first CSS framework that enables fast and responsive styling directly in your HTML or JSX code.

## Key Features

- Health Blog Articles: Access a wide range of health-related articles written to educate and inform users on various medical topics.

- Nearby Pharmacies and Hospitals : Find the nearest pharmacies and hospitals based on your current location for quick access to medical services.

- Disease and Medication Info : Browse accurate and up-to-date information about common diseases and their treatments to help users understand symptoms and available remedies.

- Search & Filter System : Easily search and filter through health content, pharmacies, hospitals, and medications to find exactly what you need.

- Bento-Style UI Layout : Modern and clean bento-style design for organizing content into visually appealing, card-like sections that are easy to scan and interact with.

## How to Run the Project Locally

If you want to try this project on your local machine, follow these steps:

1. Clone this repository  
   ```bash
   git clone https://github.com/SharkByte-Labs/Healthify.git
      ```

1. Navigate to the project folder
   ```bash
   cd healthify
   ```

1. Install dependencies 
   ```bash
   npm install
   ```

1. Start the development server
   ```bash
   npm run dev
   ```

## Brief Project Explanation

This project was created as part of the SiteFest competition with a mission to help users access trusted and easy-to-understand medical information in one place.
By using modern technologies like React Js and Tailwind CSS, this project offers a fast and responsive user experience.
Deployment with Vercel allows easy online access without complicated setup.
